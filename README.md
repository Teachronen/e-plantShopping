# e-plantShopping

## Project: Paradise Nursery Shopping Application

**Overview**
This repository, **e-plantShopping**, contains the front-end code for the "Paradise Nursery" e-commerce website. Built with React and Redux, this application provides a seamless shopping experience for houseplant enthusiasts.

**Key Features:**
* **Landing Page:** An inviting homepage with the company branding and navigation.
* **Product Listing:** A dynamic catalog of houseplants organized by category (e.g., Air Purifying, Aromatic).
* **Shopping Cart:** A fully functional cart that tracks selected items, allows quantity adjustments, and calculates total costs in real-time.
* **Global State Management:** Uses Redux to maintain the cart state across different pages.
